# OpenClaw 教程

## 什么是 OpenClaw

OpenClaw 是一个开源的 AI Agent 框架，让你可以打造属于自己的私人数字精灵。它能够：

- 🔌 **连接多种服务**：微信、Telegram、Discord 等
- 🧠 **持久记忆**：跨会话保持上下文
- 🛠️ **丰富工具**：文件操作、网页浏览、代码执行等
- 🤖 **多模型支持**：Claude、GPT、本地模型等

## 快速开始

```bash
# 安装
npm install -g openclaw

# 初始化
openclaw init

# 启动
openclaw start
```
