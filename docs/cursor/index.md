# Cursor 教程

## 什么是 Cursor

Cursor 是一款 AI 原生的代码编辑器，基于 VS Code 构建，集成了强大的 AI 编程能力。

## 主要特性

- 🤖 **AI 对话**：与 AI 自然语言交互
- ✍️ **智能补全**：上下文感知的代码补全
- 🔧 **代码编辑**：AI 辅助的代码重构
- 📝 **文档生成**：自动生成代码文档
- 🎨 **Composer**：多文件同时编辑
- 🔍 **代码库理解**：理解整个项目结构

## 快速开始

1. 下载 [Cursor](https://cursor.sh)
2. 安装并登录
3. 打开项目文件夹
4. 使用 `Cmd/Ctrl + K` 调出 AI
