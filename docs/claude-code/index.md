# Claude Code 教程

## 什么是 Claude Code

Claude Code 是 Anthropic 官方推出的命令行 AI 编程助手。它直接运行在你的终端中，能够：

- 📝 **编写和修改代码**：理解整个项目上下文
- 🐛 **调试和修复 Bug**：分析错误信息，定位问题
- 🔍 **代码审查**：检查代码质量，提出改进建议
- 📂 **文件操作**：读写文件、创建目录结构
- 🚀 **执行命令**：运行测试、构建项目、Git 操作

## 为什么选择 Claude Code

### 优势
- **纯终端体验**：不离开命令行，效率拉满
- **深度集成**：理解你的整个项目结构
- **安全可控**：代码在本地运行，敏感信息不外传
- **灵活强大**：支持自定义配置和扩展

### 适合人群
- 终端重度用户
- 注重隐私和安全的开发者
- 喜欢自动化工作流的工程师

## 快速开始

```bash
# 安装
npm install -g @anthropic-ai/claude-code

# 在项目目录启动
cd your-project
claude

# 开始对话
> 帮我分析这个项目的结构
```
