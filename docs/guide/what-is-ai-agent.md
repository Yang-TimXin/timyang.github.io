# 什么是 AI Agent

## AI Agent 的定义

AI Agent（AI 代理）是一种能够自主执行任务、做出决策并与环境交互的人工智能系统。与传统的 AI 聊天机器人不同，AI Agent 具备：

- **自主性**：能够独立规划和执行任务
- **工具使用**：可以调用外部工具和 API
- **持久记忆**：在多次交互中保持上下文
- **目标导向**：围绕特定目标持续工作

## AI Agent vs 传统 AI 助手

| 特性 | 传统 AI 助手 | AI Agent |
|------|-------------|----------|
| 交互方式 | 一问一答 | 自主规划执行 |
| 工具使用 | 无或有限 | 丰富工具生态 |
| 记忆 | 单次对话 | 跨会话持久化 |
| 任务复杂度 | 简单问答 | 多步骤复杂任务 |

## 主流 AI Agent 工具

### 编程类
- **Claude Code**：Anthropic 官方 CLI 编程助手
- **Cursor**：AI 原生代码编辑器
- **Windsurf**：Codeium 出品的 AI 编辑器

### 通用 Agent 类
- **OpenClaw**：开源 AI Agent 框架
- **AutoGPT**：自主 AI 代理
- **MetaGPT**：多 Agent 协作框架
